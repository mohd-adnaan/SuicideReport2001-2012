# Suicide Data Analysis Project

This project aims to analyze suicide data from the years 2001-2012 in India. The data provides insights into various factors like causes, age groups, and genders associated with suicides. Additionally, the project includes visualizations to better understand the trends and patterns in the data.

## Dataset

The dataset used in this project is sourced from "Suicide Report 2001-12.csv". It contains information about suicides in different states of India, including details such as state, year, cause of suicide, gender, age group, and total count.

## Contents

- `Suicide Report 2001-12.csv`: The raw dataset used for this analysis.
- `analyze_suicide_data.ipynb`: Jupyter Notebook containing the code for data cleaning, analysis, and visualization.
- `README.md`: This file.

## Project Files

- `analyze_suicide_data.ipynb`: This Jupyter Notebook contains the entire code for the project. It is divided into sections for data loading, cleaning, analysis, and visualization.
